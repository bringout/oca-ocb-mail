# Mass Mail Tests

This module contains tests related to mass mailing. Those
are present in a separate module to use specific test models defined in
test_mail. 

## Installation

```bash
pip install odoo-bringout-oca-ocb-test_mass_mailing
```

## Dependencies

This addon depends on:
- mass_mailing
- mass_mailing_sms
- test_mail
- test_mail_sms
